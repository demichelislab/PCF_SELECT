In SelectionResults bed file some amplicons covering the ERG_TMPRSS2 region are annotated with original gene names ("ETS2","IGSF5","MX2","MX1").
In the final version of the bed these genes (column 4) are renamed to "ERG_TMPRSS2" for running the method. In backup version original name are retained.
In focallog2 bed names are retained only in column 6. In backu version also chrX lines are retained.
